# Internet Propre — ULTRA HARDCORE ENGINE / Clean Internet — ULTRA HARDCORE ENGINE

**Fast page cleanup with a floating, movable UI. Choose between Normal, Hardcore, or Text-only modes to remove distractions, ads, and unwanted scripts.**  

**Nettoyage rapide des pages avec une UI flottante et déplaçable. Choisissez entre Normal, Hardcore ou Texte seul pour supprimer pubs, scripts et distractions.**

---

## English Section 🇬🇧

### Overview
Clean Internet is a lightweight Userscript designed to give you control over any webpage. It provides three modes to customize your browsing experience:

- **Normal:** Standard browsing with minimal interference.  
- **Hardcore:** Aggressively removes ads, scripts, iframes, and other unwanted elements.  
- **Text-only:** Simplifies pages to text-only format for distraction-free reading.

A floating, movable UI allows you to switch modes instantly and view a live operational log of what was cleaned.

### Features
- Floating, draggable interface  
- 3 customizable modes (Normal, Hardcore, Text-only)  
- Operational log to track actions  
- Settings saved automatically between sessions  

### Installation
1. Install [Tampermonkey](https://www.tampermonkey.net/) or a compatible Userscript manager.  
2. Add the script `internet-propre.user.js` to your manager.  
3. Open any website and use the floating UI to choose your preferred mode.  

### FAQ
**Q: Will Hardcore mode break websites?**  
A: Possibly. Hardcore aggressively removes scripts and ads, which may affect interactive content. Use with caution.  

**Q: Can I move the UI?**  
A: Yes. Simply click and drag the UI box anywhere on the screen.  

**Q: Are my settings saved?**  
A: Yes. The selected mode for each site is saved automatically.  

**Q: Which browsers are supported?**  
A: Any browser that supports Userscripts via Tampermonkey or similar extensions (Chrome, Firefox, Edge, etc.).  

### Disclaimer
Use at your own risk. Hardcore mode may break certain site functionality. This script is intended for advanced users who want a cleaner browsing experience.
